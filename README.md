웨플 wheple
===========
#### <개인프로젝트>
> 체육시설 예약 안드로이드 어플
>> 체육시설 사용자와 관리자 모두 간편한 체육 시설 예약 시스템입니다. 대학교 농구 동아리에서 활동했을 때, 체육관 대관이 복잡하고 힘들었던 경험을 바탕으로 앱을 만들어보았습니다.

<br/>

사용기술
-------
- 언어 : Kotlin, PHP
- 데이터베이스 : PostgreSQL
- API : GoogleMap, KakaoPay 
- 아키텍처 : MVP 

<br/>
